# GEMINI.md

## Directory Overview

This directory contains the source code for a simple, static website for a consultancy named "Iponion". The purpose of the website is to provide a brief overview of the company's services. It's a "short-stack" web project, meaning it uses minimal, essential web technologies.

## Key Files

*   `Index.html`: This is the main HTML file for the website. It contains the structure and content of the page, including the header, main content, and footer.
*   `style.css`: This file contains the styles for the website. It defines the layout, colors, and fonts used on the page. It also includes a media query for a dark color scheme.
*   `css2`: This file appears to be a CSS file, but it is empty. It is linked in `Index.html`.

## Usage

To view the website, open the `Index.html` file in a web browser. No special build steps or servers are required.
